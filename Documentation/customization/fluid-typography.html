# Fluid Typography

The usual technique of making your website look good on mobiles is to reduce text sizes and margins between elements as screen becomes smaller. This requires you to set different breakpoints and define how each element should look on the particular screen width.

But it’s impossible to take into consideration literally infinite number of different screens. Rhye has a cool feature for smooth tailoring text sizes depending on viewport width. It's called Fluid Typography (opens new window) It allows you to define a minimum and a maximum value for a property (for example, it can be font-size) and then it will smoothly shrink and grow depending on the screen size. Automatically. Without writing a single breakpoint rule!

# Size Presets

The typographic sizing values used throughout the template are defined as CSS variables in 📄 css/main.css under :root selector.

:root {
  ...
  /* XL Heading */
  --xl-max-font-size: 188;
  --xl-min-font-size: 54;
  --xl-line-height: 1.1;
  /* h1 Heading */
  --h1-max-font-size: 104;
  --h1-min-font-size: 35;
  --h1-line-height: 1.13;
  /* h2 Heading */
  --h2-max-font-size: 65;
  --h2-min-font-size: 31;
  --h2-line-height: 1.31;
  /* h3 Heading */
  --h3-max-font-size: 42;
  --h3-min-font-size: 24;
  --h3-line-height: 1.29;
  /* h4 Heading */
  --h4-max-font-size: 26;
  --h4-min-font-size: 22;
  --h4-line-height: 1.62;
  /* h5 Heading */
  --h5-max-font-size: 18;
  --h5-min-font-size: 18;
  --h5-line-height: 1.6;
  /* h6 Heading */
  --h6-max-font-size: 14;
  --h6-min-font-size: 14;
  --h6-line-height: 1.6;
  /* blockquote */
  --blockquote-max-font-size: 24;
  --blockquote-min-font-size: 16;
  --blockquote-line-height: 1.6;
  /* paragraph */
  --paragraph-max-font-size: 18;
  --paragraph-min-font-size: 16;
  --paragraph-line-height: 1.8;
  /* dropcap */
  --dropcap-max-font-size: 110;
  --dropcap-min-font-size: 60;
  --dropcap-line-height: 0.7;
  ...
}

# From 320px (iPhone SE screen resolution)

The minimum value with *-min-font-size suffix defines the actual font-size for the smallest screen sizes down to 320px.

# To 2560px (iMac or LG Ultrafine 5k Display screen resolution)

The maximum value with *-max-font-size suffix defines the actual font-size for the largest screen sizes up to 2560px.

# Helper Classes

Use one of the following classes to style a text element:

.xl /* XL heading */
.h1 /* h1 heading */
.h2
.h3
.h4
.h5
.h6
.strong
.em
.paragraph /* paragraph */
.has-drop-cap /* paragraph with drop cap */
.small
.small-caps

You can use the classes above in conjunction with any HTML tags. This gives a great flexibility when writing a SEO-friendly markup:

<!-- h1 heading with h1 style -->
<h1>My Heading</h1>

<!-- Still h1 heading but with h3 style -->
<h1 class="h3">My Heading</h1>

<!-- h2 heading but with XL style -->
<h2 class="xl">My Heading</h1>

<!-- Regular <span> element but with h4 style -->
<span class="h4">My Content</span>

<!-- Regular <div> element but with paragraph and drop cap style -->
<div class="paragraph has-drop-cap">My Content</h1>
