# Requirements

  • It's strongly recommended to setup a web-server first to avoid any problems with AJAX and WebGL motion effects.
  • For the best development experience it's recommended to setup a professional environment like free Visual Studio Code (opens new window)
  • The template was coded using a text editor. It was not tested with Adobe Dreamweaver or any other WYSYWIG site editor. It's recommended against the using of any visual editors.
  • The template represents a set of static HTML5 files for the further development. There is no CMS or admin panel behind and it's NOT a WordPress theme.

# Unsupported Browsers

❌ Internet Explorer
❌ Opera Mini

To improve the experience for the users with unsupported browsers it's recommended to embed Outdated Browser Rework (opens new window) into the template. This will smoothen white screen non-loading page experience. It’s a good practice nowdays since the number of visitors with old browsers tends to be lower and lower as time goes by.
